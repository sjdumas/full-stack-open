# Full Stack Open

This repository contains the assignments in the University of Helsinki [Full Stack Open](https://fullstackopen.com/en/) web development course.

The lessons in this course allows you to learn the following:

- React
- Redux
- Node.js
- MongoDB
- GraphQL
- TypeScript

The main focus is on building single page applications with ReactJS that use REST APIs built with Node.js, providing you with an introduction to modern JavaScript-based web development.
